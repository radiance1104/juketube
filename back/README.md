# JukeTube backend

## Required

Python 3.11
Node.js 25.1.0
ffmpeg
mpg123

## Installation

1. Install mpg123
  for macOS.

    ```cli
    brew install mpg123
    ```

1. Install ffmpeg
  for macOS.

    ```cli
    brew install ffmpeg
    ```

1. Install MongoDB

1. Install Node.js@10.15.3

1. Install modules
  Install modules of node.js

    ```cli
    npm install
    ```

## Start

1. Start
  default port 4300.

    ```cli
    npm start
    ```
