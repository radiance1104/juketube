# JukeTube backend

## Installation

1. Install mpg123
  for macOS.
    ```
    $ brew install mpg123
    ```

1. Install ffmpeg
  for macOS.
    ```
    $ brew install ffmpeg
    ```

1. Install MongoDB

1. Install Node.js@10.15.3

1. Install modules
  Install modules of node.js
    ```
    $ npm install
    ```

## Start

1. Start
  default port 4300.
    ```
    $ npm start
    ```
